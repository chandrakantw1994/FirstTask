Project Synopsis

Let me just brief you about the project structure

Language used: I have used JAVA Language in this Project

Framework Used: I have used TestNG testing Framework using Page Object Model Design Pattern

Packages: I have created seperated packages for Base class, Pages classes and Test class. All the web page related classes come under the Pages package and all the tests related classes come under Tests package

Extent Reports: For the reporting purpose, I have used Extent Reports. It generates beautiful HTML reports, such as index.html and emailable-report.html

Note:

  • I have used Maven as Project Management tool while creating this Poject so that I can add various dependencies and also integrated testNG dependency in pom.xml so that I can run this pom.xml file using Jenkins or command prompt, for continuos testing on a daily basis and also for nightly execution.
